## Фаза 0 — Результаты ресёрча пререквизитов

Проведён предварительный ресёрч по всем блокирующим вопросам перед стартом разработки. Полные итоги — в [[Research/00 - Prerequisites]].

### A. Unity и пакеты

- Выбрана **Unity 6.3 LTS (6000.3.x)** — текущая рекомендация Unity, поддержка до декабря 2027. Unity 6.0 отклонена (конец поддержки октябрь 2026), 6.4 отклонена (нестабильная ветка).
- Подтверждён стек пакетов: Input System 1.19, Cinemachine 3.1, AI Navigation 2.0, Test Framework 1.6, uGUI 2.0 (TMP встроен). Пакет Behavior (behavior trees) исключён — с мая 2026 в maintenance-only.
- Подтверждено: uGUI для всего UI (world-space UI Toolkit слишком нов — лишний риск).
- Твининг: PrimeTween (MIT, zero-alloc) — вместо DOTween (closed-source, NRE-ловушки).

### B. Аудио-пайплайн

Все облачные сервисы (Suno, Udio, ElevenLabs, Stable Audio API, Azure TTS) исключены ограничением нулевого бюджета. Выбраны:
- Музыка: incompetech/Kevin MacLeod (CC-BY 4.0) + Pixabay + OpenGameArt.
- SFX: Kenney Sci-fi Sounds (CC0) + Sonniss GDC 2026 (royalty-free).
- Голоса юнитов: Kenney Voiceover Pack + Voiceover Pack (Fighter) — CC0, ноль усилий.
- Локальная генерация (опционально): Stable Audio Open (открытые веса), Piper TTS (CPU).

### C. Арт / визуал

Платные паки (Synty POLYGON) исключены по нулевому бюджету и неподходящей EULA (нельзя в открытый репо). Выбраны:
- Юниты: Quaternius Animated Mech Pack + Animated Tanks Pack (CC0).
- Здания: Quaternius Ultimate Modular Sci-Fi (CC0).
- Пропсы/оружие: Kenney Blaster Kit + Space Kit (CC0).
- UI: Kenney UI Pack Sci-Fi (CC0) + game-icons.net (CC-BY 3.0, атрибуция в Credits).
- Шрифты: Russo One + Exo 2 (OFL, кириллица есть).

### D. VCS и CI

- LFS настраивается до первого коммита, выборочно: крупные бинарники в LFS, мелкие png/ogg — обычный git (ограничение: 10 GiB/мес трафика на free-tier).
- Релизы — GitHub Releases (2 GiB/файл, трафик бесплатен).
- CI (GameCI v4 + GitHub Actions): Actions для публичных репо бесплатны, но нужен секрет `UNITY_LICENSE`. **Решение: стартуем без CI**, добавим после M0 по желанию пользователя.
