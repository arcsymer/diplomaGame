## WebGL (ADR-027)

**Цель:** защита диплома без установки — игра открывается по ссылке.

**Реализация:** Forge «Build WebGL (v8)» / `ForgeBatch.BuildWebGL`; Gzip +
decompressionFallback (статический хостинг без заголовков); guard файловой системы
в ScreenshotDirector. Публикация: ветка gh-pages → GitHub Pages; zip — в релиз.
itch.io — вручную владельцем аккаунта (тот же zip).
