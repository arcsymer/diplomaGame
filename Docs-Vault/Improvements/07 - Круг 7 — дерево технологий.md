## Дерево технологий (ADR-026)

**Benchmark (SC2):** апгрейды исследуются в зданиях через общую очередь производства,
занимают слот, дают глобальные для фракции бонусы; кнопки в command card с
индикатором исследованного и гейтингом по prerequisites.

**Дерево:**

```
[Tier-1] Бронирование      +20% HP пехоте       150 / 40 с   [R]
[Tier-1] Усиленные стволы  +15% урона всем      175 / 45 с   [G]
[Tier-2] Расширенные обоймы −15% кулдаун атаки  200 / 50 с   [X] ← требует Стволы
```

**Макро-решение:** Бронирование ≈ цена 3 Marines — окупается при армии 5+;
ИИ-противник исследует по тому же дереву (баланс ≥ цена + резерв на юнита) —
сложность растёт без роста микро.

**Архитектура:** TechData (SO) + TechRegistry (per-faction, кэш множителей без
аллокаций в кадре) + ProductionEntry.techData (общая очередь, legacy-совместимость)
+ второй ряд command card. Урон/кулдаун — лениво, MaxHp — только новым юнитам
(SC2-семантика). Forge: «Create/Update Tech Assets (v7)» + ForgeBatch.SetupTechTree.
