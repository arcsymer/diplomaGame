## Лэйаут (ADR-028)

**Benchmark (SC2):** карта 1v1 — точечная симметрия, главные базы, естественные
экспанды (риск/награда), чокпоинты, фланговые обходы.

**Было:** пустой квадрат, базы по углам, одна линия столкновения.
**Стало:** базы (±35,±35) → центральный чокпоинт (проход ~11 ед., 6 скал,
NavigationStatic-бейк) → фланги открыты → по экспанд-месторождению ×2000 на сторону
ближе к центру → 15 CC0-декораций. Три пути атаки с разным риском; экспанд игрока
работает через существующий BuildingPlacer/Extractor.
