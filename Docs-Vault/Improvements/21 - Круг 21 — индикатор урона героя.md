# Круг 21 — индикатор урона героя (red flash)

Дата: 2026-06-23. Ветка `improve/circle-21` → main, тег `v1.21.0`.

## Improvement: фидбек получения урона героем
Замыкает петлю боевого фидбека героя: круг 20 = «я попал» (хитмаркер), круг 21 = «попали в меня». До этого получение урона героем не давало экранного сигнала (только тик HP-бара).
- **Что:** при уроне герою — красная вспышка по краю экрана (vignette), затухание ~1 с. На TPS_Block (авто-скрытие в RTS-режиме через OnDisable).
- **Честное ограничение:** `Health.AnyDamaged` = `static Action<Health, float>` — НЕ несёт позицию источника урона. Направленный индикатор (стрелка к атакующему) невозможен без изменения API Health → реализована полноэкранная вспышка (fallback по спеке, без фейковой «направленности»). Направленный вариант — бэклог (нужен Health с позицией источника; чистая логика `HeroDamageIndicatorLogic` уже готова к расширению углом).
- **Код:** `HeroDamageIndicatorLogic` (ComputeFadeAlpha / IsFadeDone, 10 тестов) + `HeroDamageIndicator` (подписка Health.AnyDamaged, фильтр по герою + !IsDead, interrupt-restart корутина, Time.unscaledDeltaTime, raycastTarget=false). 2 поля в GameFeelSettings (duration 1.0, peakAlpha 0.6). Forge `SetupHeroDamageIndicator` (+ ForceReserializeAssets для персиста полей — урок круга 20 применён превентивно).
