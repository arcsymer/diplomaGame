## Принципы приоритизации

1. Сначала — разрывы DoD промпта v2 (честность диплома): способности, тултипы.
2. Затем — измерительная инфраструктура v3 (без неё нельзя верифицировать улучшения).
3. Высокий impact / низкий effort — вперёд; высокий effort — только при высоком impact.
4. Нулевой бюджет фильтрует кандидатов до FOSS/CC0 (ADR-009).
